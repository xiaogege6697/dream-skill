# 贡献指引（CONTRIBUTING）

dream skill —— 模仿人脑睡眠整理文件记忆库的工作流。欢迎贡献。

## 反馈
提 issue：说明触发场景 + 期望 + 实际。

## 改进 skill
1. Fork → 改 SKILL.md / references → 提 PR
2. PR 说明：改了什么、为什么

## 设计原则（改动请遵守）
- **保守修剪**：信息丢失代价不对称，高危删除等用户确认
- **只整理不创造**：dream 不编造新知识，判定主线程把关
- **不 cron**：偶尔提醒 + 用户触发，不自动跑
- **transcripts 红线**：原始层只读，永不删/移/改

## 本地测试
clone 到 `~/.claude/skills/dream/`，说「做梦 / dream」触发，跑一遍验证流程。
